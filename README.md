# model-fit-estimates repository

2026-02-04

This repository provides the different models used in the framework of WHO's Foodborne Disease Burden Epidemiology Reference Group (FERG) towards the global burden of foodborne disease. The repository contains models used to estimate the incidence/prevalence/case-fatality rate/mortality for different hazards.



The full methodology is outlined in *Methodological framework for the World Health Organization estimates of the global, regional and national burden of foodborne diseases 2026 edition*, submitted to xx. All FERG estimates have been published as part of a dedicated xx.
